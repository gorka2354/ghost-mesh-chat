# Ghost Mesh Chat

**Анонимный P2P мессенджер** — без серверов, без регистрации, без следов.

---

## Концепция

Браузерное приложение (PWA) для анонимного общения напрямую между устройствами через WebRTC. Сообщения не проходят через серверы — только peer-to-peer. Нет аккаунтов, нет базы данных, нет истории на сервере.

---

## Ключевые фишки

| Фишка | Описание |
|---|---|
| **Без серверов** | Сообщения идут напрямую между браузерами (WebRTC) |
| **Без регистрации** | Открыл → получил ID → отправил ссылку → общаешься |
| **Mesh-сеть** | Каждый участник подключён к каждому, нет центральной точки |
| **Самоуничтожение** | Закрыл вкладку — чата больше нет. Никакой истории |
| **PWA** | Устанавливается как приложение на телефон и десктоп |
| **Шифрование** | WebRTC использует DTLS/SRTP — трафик зашифрован по умолчанию |

---

## Стек

- **Frontend:** HTML + CSS + vanilla JS (или React если захочется)
- **WebRTC:** PeerJS (обёртка над WebRTC, упрощает подключение)
- **Signaling:** PeerJS Cloud (бесплатный) или свой signaling-сервер
- **PWA:** manifest.json + service worker
- **Опционально:** Tauri (десктоп), Capacitor (мобильное приложение)

---

## Архитектура

```
┌──────────┐         PeerJS Cloud          ┌──────────┐
│  Peer A  │ ──── signaling (ICE/SDP) ────▶│  Peer B  │
│ (браузер)│                               │ (браузер)│
└────┬─────┘                               └────┬─────┘
     │          WebRTC DataChannel              │
     └──────── прямое соединение ───────────────┘
               (зашифрованное, P2P)
```

**Signaling-сервер** нужен только для начального рукопожатия (обмен ICE-кандидатами). После установки соединения — всё идёт напрямую.

---

## Сценарии использования

1. **Быстрый анонимный чат** — скинул ссылку, поговорили, закрыл
2. **Групповой mesh-чат** — до ~10 человек в одной комнате
3. **Передача файлов** — drag & drop файл → отправка через DataChannel
4. **Экстренная связь** — когда нужен чат без следов и регистрации

## Чем отличается от существующих

| Проект | Минус |
|---|---|
| Telegram | Серверы, аккаунт, номер телефона |
| Signal | Номер телефона, серверы |
| Briar | Только Android, сложный |
| Session | Блокчейн, медленный |
| **Ghost Mesh** | Браузер, без регистрации, мгновенно |
